# azure-isucon-templates

This template allows you to create a Virtual Machines for [ISUCON](http://isucon.net/).

- ISUCON1
- ISUCON2
- ISUCON3-qualifier
- ISUCON3-final
- ISUCON4-qualifier
- ISUCON4-final
- (ISUCON5-qualifier](https://github.com/matsuu/azure-isucon-templates/tree/master/isucon5-qualifier)
- ISUCON5-final
